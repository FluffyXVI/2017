January

Pixelception

A series of scripts that generate an image from a collage of different images. These include generating a palette from a folder of images, and matching each pixel in the base image to a palette sample.

Mandelbrot

Originally created for a Maths C assignment in 2016, then updated in January by request of a friend. These scripts focus on analysing variations of the Mandelbrot set and Julia sets, and explore some of the correlations between them.

February

MicroPoint

Experiments in using data from a micro:bit to control the mouse cursor, or to update a visual display.

Solar System

A remake of a project originally completed in 2015. A simplified simulation of a solar system, including stars, planets, and moons.
